- `dotnet-backend-expert` agent for ASP.NET Core, Wolverine CQRS, and MartenDB event sourcing
- Agent dynamically loads the dotnet-framework skill from packages/blazor/skills/dotnet-framework/
- Skills directory placeholder for future .NET-specific skill additions
