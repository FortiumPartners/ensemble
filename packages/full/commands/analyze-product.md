Analyze a product or feature idea to understand requirements and plan implementation.

## Mission

Perform comprehensive product analysis including:
- Market and competitive analysis
- User needs assessment
- Technical feasibility evaluation
- Implementation complexity estimation

## Workflow

1. **Discovery**
   - Understand the product/feature concept
   - Identify target users and use cases
   - Research similar solutions

2. **Analysis**
   - Assess technical requirements
   - Evaluate implementation approaches
   - Identify risks and dependencies

3. **Output**
   - Product analysis report
   - Recommended next steps

## Usage

```
/ensemble:analyze-product <product or feature description>
```

Provides foundation for PRD creation or technical planning.
