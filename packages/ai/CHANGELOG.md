# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [5.1.0] - 2025-12-31

### Added
- Initial release of ensemble-ai package
- OpenAI skill with comprehensive SDK documentation
- GPT-5 family model coverage (GPT-5, GPT-5.1, GPT-5.2)
- Chat Completions API patterns
- Responses API patterns
- Agents SDK integration
- Embeddings API (text-embedding-3-small/large)
- Tool calling and function patterns
- Streaming patterns (sync and async)
- Templates for Python and TypeScript
- Production-ready examples
- Progressive disclosure documentation (SKILL.md and REFERENCE.md)
- Automatic framework detection based on dependencies
- Multi-language support (Python, TypeScript)

### Documentation
- Quick reference guide (SKILL.md) for fast lookups
- Comprehensive reference (REFERENCE.md) for deep dives
- Template usage guide with placeholders
- Example implementations with best practices
- Validation matrix for feature coverage
